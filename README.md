## 📋 Sobre o projeto

O **RP Transportes** é um site institucional criado para divulgar os serviços e a área de cobertura da empresa. A página conta com seções de apresentação, serviços oferecidos, regiões atendidas e formulário/contato, além de integração com WhatsApp para atendimento rápido.

**Em breve**, o site contará com um **dashboard** para o cliente poder **personalizar as informações do site** (textos, imagens, contatos e serviços) de forma autônoma, sem depender de alterações técnicas.

## 🚀 Como rodar

```bash
# Clonar o repositório
git clone <url-do-repositorio>
cd rp-transportes-site

# Instalar dependências
npm install

# Rodar em desenvolvimento
npm run dev

# Build para produção
npm run build

# Preview do build
npm run preview
```
